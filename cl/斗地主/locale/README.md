# locale 目录说明

这个目录放 Factorio 的本地化文本。

## 目录结构

- `zh-CN/locale.cfg`：简体中文文本。
- `en/locale.cfg`：英文文本。

## 确定规则

- 客户端语言是 `zh-CN`，就加载 `zh-CN/locale.cfg`
- 客户端语言是 `en`，就加载 `en/locale.cfg`
- 客户端语言是 `zh-TW`，且没有 `zh-TW/locale.cfg` 时，回退到 `en`

## 用途

`locale.cfg` 里的键会被游戏自动读取，用来显示：

- 窗口标题
- 按钮文字
- 提示文本
- 规则说明
- 牌面测试工具文案
- 音效测试文案

## 规则

- 两个语言目录尽量保持同一套键名。
- 中文和英文只需要翻译值，不要改键名。
- 这里不放 Lua 逻辑，只放可翻译文本。

## 说明

当前项目的主语言是 `zh-CN`，`en` 作为英文对照。
