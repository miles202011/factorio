# AI 参考源码目录

这个目录专门放斗地主相关的外部参考源码，保留原始发布包和解压后的目录结构，方便对照源码和追溯来源。

## 目录作用

- `DouZero-1.1.0/`：DouZero 的本地参考源码。`DouZero.md` 里的源码路径都指向这里，用来参考出牌、候选生成、评估和 DMC 相关逻辑。
- `rlcard-1.0.7/`：RLCard 的本地参考源码。主要作为斗地主/卡牌环境实现的辅助参考，不是 Factorio 直接运行的代码。

## 压缩包

- `DouZero-1.1.0.zip`：DouZero 原始下载包，保留作来源归档。
- `rlcard-1.0.7.zip`：RLCard 原始下载包，保留作来源归档。

## 使用方式

- 需要对照 DouZero 时，优先看 `DouZero-1.1.0/DouZero-1.1.0/` 下面的源码。
- 需要看 RLCard 的斗地主环境、规则或训练接口时，优先看 `rlcard-1.0.7/rlcard-1.0.7/` 下面的源码。
- 这些目录不是游戏运行时直接加载的资源，只用于整理和参考。
