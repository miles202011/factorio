# 井字棋

## 文件性质

这是 Factorio 场景替换文件 `control.lua`，不是 mod。脚本放在独立文件夹 `cl/井字棋/` 下，第一行加载 freeplay：

```lua
require('__base__/script/freeplay/control.lua')
```

## 事件注册方式

使用 `require("event_handler")` 后通过 `handler.add_lib(...)` 注册事件和初始化逻辑，避免直接覆盖 freeplay 已注册的事件处理器。

注册事件：

- `on_player_created`：给新玩家添加顶部按钮。
- `on_player_joined_game`：玩家加入时补加顶部按钮。
- `on_player_left_game`：玩家在对局中离线时，自动判另一方获胜。
- `on_pre_player_removed`：玩家被移除前清理棋桌状态。
- `on_gui_click`：处理大厅、棋桌、棋盘落子、准备、离桌等按钮。

## 核心逻辑

- 顶部按钮名为 `ttt_top_btn`，标题为“井字棋”。
- 玩家点击按钮打开大厅。
- 大厅可开新桌、加入等待中的棋桌、查看已有棋桌。
- 每桌两席，座位 1 为 `X`，座位 2 为 `O`。
- 等待阶段双方准备后开始。
- 单人可点击“补 AI”，AI 自动准备并参与对局。
- 等待阶段桌内有 AI 时可点击“踢 AI”，移除 AI 并继续等待真人或重新补 AI。
- 对局阶段按 3x3 棋盘按钮落子。
- 真人玩家每回合有 20 秒落子时间，超时未落子则判对方获胜。
- 棋盘上的 `X` / `O` 使用更大的棋子样式显示；已落子的格子不走禁用态，避免棋子被压成小灰字。
- 胜负判断包含 8 条标准三连线。
- 棋盘填满且无人三连时判平局。
- 游戏结束后可点击“再来一局”，重置棋盘和准备状态。
- 战绩按真人玩家名记录胜、负、平；AI 不记录战绩。

## 关键数据表

持久化入口：

```lua
storage.ttt = {
  tables = {},
  seat = {},
  next_tid = 1,
  stats = {},
  ai_due = {},
  turn_due = {},
}
```

棋桌结构：

```lua
{
  tid = 1,
  phase = "waiting" | "playing" | "over",
  players = {[1] = player_index_or_ai, [2] = player_index_or_ai},
  ready = {[player_index_or_ai] = true},
  board = {"", "", "", "", "", "", "", "", ""},
  turn = player_index_or_ai,
  winner = player_index_or_ai,
  msg = "状态提示",
}
```

说明：

- 真人玩家使用正数 `player.index`。
- AI 使用负数 `-tid`。
- `storage.ttt.seat[player_index] = tid` 用于快速定位玩家所在棋桌。
- `storage.ttt.stats[player_name] = {w=0, l=0, d=0}` 记录真人战绩。
- `storage.ttt.turn_due[tid] = tick` 记录当前真人回合的 20 秒倒计时截止 tick。

## AI 策略

AI 不会在玩家落子后立刻行动，而是延迟约 1 秒再落子，避免看起来像“秒出牌”。

AI 每回合按以下优先级选格：

1. 自己能立即三连则落子取胜。
2. 对手能立即三连则阻挡。
3. 抢中心。
4. 抢角。
5. 选择第一个空格。

## 注意事项

- 此脚本只新增 `cl/井字棋/control.lua` 和 `README.md`，不会写入 Factorio 安装目录或用户存档目录。
- 如果要部署到场景目录，需要在用户明确要求“同步”或“复制过去”后再操作。
- 当前 UI 以轻量可玩为目标，没有使用额外贴图或自定义样式。

## HTML 版

- `index.html` 是同功能浏览器版，复刻 `control.lua` 的顶部按钮、大厅、棋桌、准备、补 AI、踢 AI、落子、胜负/平局、再来一局、离桌和战绩。
- HTML 版不依赖 Factorio，可直接用浏览器打开。
- 浏览器版用“当前玩家”按钮模拟多名玩家切换。

## 面板风格

- 面板结构参考 `cl/斗地主`：自定义标题栏 + 可拖拽空白区 + 右上角关闭按钮。
- 大厅使用牌桌卡片列表，底部提供「开新桌」「查看战绩」「游戏规则」「注意事项」。
- 等待页显示「玩家就绪 (n/2)」、逐座位准备状态、加入 AI、踢出 AI、离桌、规则和注意事项。
- 对局页显示两张玩家信息卡，当前回合玩家高亮，棋盘居中，底部提供规则和「退出（结束本局）」。
- 结束页显示结果，并提供「再来一局」「离桌」「大厅」。
- 大厅不直接显示当前玩家战绩，战绩统一通过「查看战绩」窗口查看。
- AI 补位说明放在「注意事项」，不放入「游戏规则」。
